K RP Game Engine

Python-based engine for rapid prototyping 2D games!

Download as .zip Download as .tar.gz View on GitHub

Keerthik's Rapid Prototyping (Game) Engine (KRPE).

Built in python using pygame, KRPE is designed for fast testing of mechanics and to circumvent the details of making a game functional so you can get to implementing core algorithms and systems.

Things You Don't Have to Worry About

  • Game Loop
  • Pausing/Unpausing
  • Multi-level menus
  • Game Object tracking
  • Sprite management
  • Game window set up
  • Mouse interactions like click, drag-n-drop, right-click

What YOU Do

  • Populate your menus
  • Create update logic for your game objects
  • Supply assets
  • Override anything you want because PYTHON
  • Use base pygame operations whenever necessary
